# SceneTalk 英语口语陪练工具
一款**沉浸式场景化、带AI上下文记忆**的英语口语对话练习网页，搭配语音交互与双语翻译，提供一站式口语练习体验。  
当前版本为**可完整演示的MVP雏形**，已打通前后端交互。

---

## ✨ 核心亮点
- 🎭 **场景化对话**：内置高频生活场景，AI角色化回复，告别生硬对话
- 🧠 **连贯记忆**：基于Chroma实现对话上下文记忆，交流自然不脱节
- 🔊 **全语音交互**：支持语音输入+AI语音播报，还原真实口语交流
- 📖 **双语辅助**：实时中英对照翻译，边练边学，理解无压力
- 📱 **轻量化体验**：纯网页运行，无需安装，打开即用

---

## 🎯 核心功能清单
### 1. 多场景模拟练习
- 内置**餐厅点餐 / 英语面试 / 酒店入住**三大高频生活场景
- 每个场景配备专属AI角色（服务员/面试官/前台）与定制开场白
- 场景切换自动重置对话，避免不同场景内容串场

### 2. AI智能对话记忆
- 完整保留对话历史，AI基于上下文连贯回复，告别机械式单轮对话
- 场景隔离记忆机制，切换场景自动清空历史，保证对话真实性

### 3. 双模式输入交互
- **语音输入**：调用麦克风实时录音，还原口语交流场景
- **文字输入**：键盘输入英文快速发送，适配安静/不方便说话的练习场景

### 4. 语音播报与双语翻译
- AI回复自动英文语音朗读，全程发音示范，纠正听力偏差
- 对话内容同步中英对照翻译，左侧对话+右侧翻译双栏展示，边练边理解

### 5. 简洁易用的界面交互
- 对话区+翻译区对称布局，界面整洁直观
- 场景一键切换、聊天气泡样式、对话自动滚动，交互流畅
- 录音按钮智能切换（开始/结束发送），操作便捷无门槛

---

## 🛠️ 技术栈
- **前端**：Web交互界面（场景选择、聊天窗口、语音/文字输入模块）
- **后端**：AI对话服务接口，支撑全链路交互逻辑
- **核心依赖**：Chroma（对话记忆）、语音识别/合成服务、AI对话Agent

---

## 🚀 快速开始
1. 克隆项目到本地
```bash
git clone <你的仓库地址>
cd SceneTalk
```
2. 启动后端服务（按你的项目启动方式执行）
3. 打开前端网页，选择场景开始口语练习
