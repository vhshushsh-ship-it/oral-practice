# AI Oral Coach
AI 智能英语口语陪练助手 | 全链路语音交互，沉浸式场景化口语练习

## ✨ 核心功能
- 🔊 **语音输入**：支持实时语音对话，解放双手自然交流
- 🤖 **AI对话**：智能对话Agent，流畅口语交互
- 🔈 **语音输出**：AI实时语音播报回复，沉浸式口语体验
- 🎭 **场景模拟**：多场景自由切换，贴合真实口语使用环境
- 🚪 **可控退出**：支持安全退出对话，操作灵活便捷

## 🌟 技术亮点
- ✅ 口语内容**自动纠错**，实时纠正语法/表达问题
- ✅ 基于 **Chroma** 实现对话记忆，上下文连贯不中断
- ✅ 轻量化架构，开箱即用，可直接演示

## 📌 项目说明
本项目为**可演示的产品雏形**，专注解决英语口语练习痛点，打造随时随地的AI口语陪练工具。

## 🚀 使用方式
1. 启动项目
2. 选择练习场景
3. 开启语音输入，与AI对话练习
4. 随时触发指令退出对话
